SSH 配置文件解析


sshd_config

  • 解决SSH登陆缓慢
    • 编辑SSHD_CONFIG文件，将UseDNS改为no， 关闭GSSAPIAuthentication no
    • 检查/etc/hosts 查看ip映射
  • Server限制Ip登陆
    • /etc/hosts.deny 设置为 sshd: ALL: deny
    • /etc/hosts.allow 设置ip访问： sshd : x.x.x.x : allow
