生成SSH Key

在shell中键入以下命令:ssh-keygen -t rsa -C "your_email@example.com"

将会提示输入存放秘钥的文件名。以及密码。

命令完成后会生成两个文件 id_rsa私钥和id_rsa.pub公钥，将其放置于 ~/.ssh/ 目录下。
